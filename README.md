# Welcome to your Lovable project

📑 Documentação Técnica: Projeto Cronos Narrative InsightStatus do Projeto: MVP Funcional com Integração de IA e Camada de Cache. Stack Tecnológica: React (Vite), Tailwind CSS, Supabase (Edge Functions & DB), Mistral AI (Large 2512). 
1. Visão Geral da ArquiteturaO sistema funciona como um motor de inteligência financeira que correlaciona dados de mercado em tempo real com um histórico econômico de 20 anos.  A arquitetura é composta por:Frontend: Interface mobile-first desenvolvida em React, focada em legibilidade e autoridade visual.Backend (Supabase Edge Functions): Centraliza a lógica de processamento e segurança. Camada de Dados: Tabelas no Supabase que gerenciam desde o histórico de mercado até o cache de consultas da IA.
2. Componentes e Funcionalidades (O que já foi feito)Interface e Experiência do Usuário (UI/UX)Autópsia Narrativa: Componente central que exibe a análise profunda do ativo selecionado.Sparklines Minimalistas: Indicadores visuais de tendência de preço acima dos títulos das análises.Smart Chips de Consulta: Atalhos para perguntas guiadas (ex: "O passado rima?", "Cabo de guerra") com respostas em componentes de Accordion.Seletor de Ativos: Navegação fluida entre IBOV, Dólar, SELIC e IPCA.Termômetro de Sentimento: Interação de engajamento baseada em emojis para capturar o sentimento do usuário.Inteligência e BackendIntegração Mistral AI: Implementação da API Mistral Large 2512 para geração de respostas baseadas em perfis de investidores. Sistema de Cache (Tabela user_analytics): Otimização de latência onde respostas anteriores são armazenadas e retornadas em menos de 2 segundos para consultas repetidas. Injeção de Contexto: A IA consome os últimos 15 registros de dados de mercado da tabela market_data para fundamentar as análises.
3. Requisitos de Sistema e Regras de NegócioRequisitos Funcionais (RF)RF01: O app deve alternar o contexto de análise conforme o ativo selecionado pelo usuário.RF02: Deve haver uma "Escada de Resposta" que adapte o tom técnico conforme o perfil do investidor (Iniciante a Experiente). RF03: Integração obrigatória com links de afiliados dinâmicos no campo de CTA (Call to Action).Requisitos Não Funcionais (RNF)RNF01: Tempo de resposta da IA (com cache) deve ser inferior a 2s. RNF02: Segurança de chaves via variáveis de ambiente (VITE_SUPABASE_URL e VITE_SUPABASE_ANON_KEY).RNF03: Design responsivo absoluto, priorizando a visualização em dispositivos móveis.
